# sizeof()和length(),strlen()

在 C++ 中，`sizeof` 运算符用于获取对象或数据类型的大小（以字节为单位），而不是获取字符串的长度。所以，使用 `sizeof(a)` 得到的结果是 `string` 类型的对象 `a` 所占用的字节数。

可以使用`a.length()`来获取字符串a的长度



`strlen(a)`不能应用于string类型的对象,因为strlen需要以`\0`为结尾
